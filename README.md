# SwasthyeDip Foundation Coming Soon Page

Static under-construction demo for a nonprofit health organization. It is ready for GitHub Pages because it uses only HTML, CSS, and a local image asset.

Target domain: `swasthyedip.org.np`

## Files

- `index.html` - page markup
- `styles.css` - responsive styling
- `assets/community-health-hero.png` - local hero image

## Publish on GitHub Pages

1. Push these files to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select the branch, usually `main`, and folder `/root`.
6. Save and wait for GitHub to publish the site.

The contact and update form use `mailto:` links for demo purposes. For real signups, connect a form service such as Formspree, Netlify Forms, or a small backend.
